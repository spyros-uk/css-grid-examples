# CSS Grid Examples

A number of CSS Grid Examples to help understand how to set-up a grid layout using CSS.

![CSS Grid Examples Screenshot](.github/screenshot.png)

## Getting Started

1. Clone the repo
2. Run `npm start`
3. Un-comment the desired example in the `src/index.js` file
